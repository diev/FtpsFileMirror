# FtpsFileMirror

Downloads all files from a remote FTPS server (i.e. MOEX) to a local mirror.

## Requirements

- .NET 7.0
- FluentFTP

See also [FTPSReportsDownloader] for just a good worker.

- .Net Framework 4.8
