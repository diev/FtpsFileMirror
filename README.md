# FtpsFileMirror

Downloads all files from a remote FTPS server (i.e. MOEX) to a local mirror.

## Requirements

- .NET 7.0
- FluentFTP

# Latest releases

FtpsFileMirror currently downgraded to old [FTPSReportsDownloader]
with requirements

- .Net Framework 4.8
